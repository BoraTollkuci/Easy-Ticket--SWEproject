# Easy-Ticket--SWEproject
 Application Overview

Easy Ticket Albania is a digital ticketing platform designed to simplify the process of booking and managing bus transportation tickets in Albania. The application allows passengers to easily search for routes, view available schedules, select seats, and purchase tickets online.

After completing a booking, the system generates digital tickets with QR codes, which can be downloaded as images or PDF files. These QR codes can be scanned by conductors during the trip to verify ticket validity.

The platform includes three main user roles:
Passenger (User),
Conductor and
Admin.

The goal of the application is to digitalize the ticket booking process, reduce manual ticket handling, and provide a fast and convenient travel experience for passengers and transport operators.
